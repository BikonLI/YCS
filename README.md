# 🎯 YCS 图形化界面参数调节说明（中文说明文档）

> 本文适用于 YCS 辅助系统的图形化设置界面，所有参数均已中文化。调参不求人，看这一个就够！

---

## 🧠 系统运行相关参数

| 中文名称    | 说明                                  | 建议          | 对应命令行参数      |
| ------- | ----------------------------------- | ----------- | ------------ |
| 🔁 刷新   | 每帧检查一次系统状态和HID设备连接，时间 >25ms 会导致瞄准震荡 | 无需手动设置，仅供观察 | *(自动)*       |
| 🎯 辅助瞄准 | 启用或关闭辅助瞄准功能                         | ✅ 推荐开启      | `--aimLock`  |
| 🧩 启用硬件 | 使用硬件（外部设备）进行鼠标输入模拟                  | ✅ 使用设备时开启   | `--hardware` |

## 👁️‍🗨️ 显示选项

| 中文名称      | 说明             | 建议        | 对应命令行参数            |
| --------- | -------------- | --------- | ------------------ |
| 📦 显示区域框  | 显示识别区域（调试用）    | 可选        | `--showAreaBox`    |

| 🧠 显示人员头像 | 显示人物头部关键点（鼻尖处） | ✅ 推荐开启调试用 | `--showPeopleHead` |

## 📷 识别区域参数

| -------- | ---------------- | ----- | ---------- |
| 📏 识别框宽度 | 图像识别区域的宽度（单位 px） | 默认550 | `--width`  |
| 📏 识别框高度 | 图像识别区域的高度        | 默认550 | `--height` |

## 🔫 自动操作参数

| 中文名称          | 说明                              | 建议           | 对应命令行参数         |
| ------------- | ------------------------------- | ------------ | --------------- |
| 🔫 自动射击       | 瞄准成功后自动射击                       | 可选，配合宏或射击键使用 | `--shootOnAim`  |
| ⌨️ 控制键的 VK 键码 | 触发辅助瞄准的热键，填写虚拟键码（VK）值，如 5 为鼠标中键 | 推荐使用鼠标侧键/中键  | `--virtualKey`  |
| 🔄 控制模式       | 0：按住触发，1：点击切换                   | 默认 0         | `--controlMode` |

---

## 🧠 识别置信度设置

| 中文名称       | 说明                  | 推荐值           | 对应命令行参数            |
| ---------- | ------------------- | ------------- | ------------------ |
| ✅ 识别框置信度阈值 | 越高越精准，越低识别得越多但容易误识别 | `0.60 ~ 0.70` | `--scoreThreshold` |
| 👥 人员重合度阈值 | 用于去除重复识别的人物框（IOU阈值） | 默认 `0.35`     | `--iouThreshold`   |

## 🧮 PID 控制参数（精准瞄准核心）

| 中文名称 | 说明                 | 推荐           | 对应命令行参数 |
| ---- | ------------------ | ------------ | ------- |
| `Px` | x 轴 PID 的 P 值（比例项） | `0.3~0.6`    | `--Px`  |
| `Ix` | x 轴 PID 的 I 值（积分项） | `0.005~0.02` | `--Ix`  |
| `Dx` | x 轴 PID 的 D 值（微分项） | `0.03~0.1`   | `--Dx`  |
| `Py` | y 轴 PID 的 P 值      | 同上           | `--Py`  |
| `Iy` | y 轴 PID 的 I 值      | 同上           | `--Iy`  |
| `Dy` | y 轴 PID 的 D 值      | 同上           | `--Dy`  |

🧪 **调参建议**：

| 问题现象      | 调整方法                          |
| --------- | ----------------------------- |
| 🎯 准心来回抖动 | 降低 `P` 或增加 `D`                |
| 🎯 准心一直飘走 | 降低 `I`，增大 `D`                 |
| 🎯 准心反应太慢 | 增大 `P`                        |
| 🎯 一甩就甩过头 | `P` 太大 or `maxAimStep` 太大，需下调 |

---

## 🖱️ 鼠标行为控制参数

| 中文名称       | 说明                    | 推荐            | 对应命令行参数        |
| ---------- | --------------------- | ------------- | -------------- |
| ➖ 鼠标移动最小步长 | 如果准星和目标之间距离小于该值，则不再移动 | 默认 `1.0`      | `--minAimStep` |
| ➕ 鼠标最大移动步长 | 限制单次鼠标移动最大值，防止画面“闪跳”  | `15.0 ~ 20.0` | `--maxAimStep` |

---

## ⚙️ 程序周期控制参数

| 中文名称        | 说明                               | 推荐值               | 对应命令行参数                |
| ----------- | -------------------------------- | ----------------- | ---------------------- |
| ⏱️ 鼠标控制输入周期 | 单位：ms。值越小，鼠标越频繁操作（震荡风险↑）         | `0.5` 以上（太低可能不稳定） | `--mouseInputInterval` |
| 🌀 总轮询周期    | 单位：ms。建议高性能电脑设置为 `6~10`，低端电脑保留默认 | 默认 `2`            | `--minInterval`        |

| ------- | ----------------------- | ------------ |
| 🧪 测试模式 | 不会真实开枪，仅用于观测识别效果、鼠标移动路径 | `--testMode` |

## 📌 示例推荐配置（进阶玩家）

```bash
# 高性能电脑 + 开启硬件 + 自动射击 + 自定义 PID + 最大帧率 + 鼠标平滑
ycs --aimLock --hardware --shootOnAim --Px 0.3 --Ix 0.01 --Dx 0.08 --Py 0.3 --Iy 0.01 --Dy 0.08 --mouseInputInterval 0.5 --minInterval 6
```

## 🧼 温馨提醒

* 参数调整**请务必使用测试模式先行试验**
* 不建议使用极限参数（如 `Px > 1.0`, `步长 > 30`）避免造成画面异常
* 推荐使用 16:9 分辨率的游戏模式，窗口/全屏皆可
* 请勿用于违规用途，**本系统仅为开发研究演示用途**
